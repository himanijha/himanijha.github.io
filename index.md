Things I'm interested in:

* **Systems and performance.** Last summer at Pure Storage, I designed a kernel bypass data path to optimize performance for a block storage driver supporting a major AI hyperscaler. I replaced the asynchronous io_uring interface with a userspace, zero copy shared memory API in C++ and discovered bottlenecks using the perf tool. I'm currently continuing in this direction through systems research at UCLA, working on similar questions around performance and efficiency at the systems level.
* **Mobile development.** I built [BinBot](https://github.com/himanijha/BinBot/tree/main), an iOS app that helps people sort their trash using a custom Create ML model. I also made PianoTutor, an Augmented Reality application built in Swift Playgrounds, which won **[Apple's Swift Student Challenge](https://developer.apple.com/swift-student-challenge/)**.
* **Accessibility.**
  * PianoTutor encodes rhythm and timing directly into its arrows pointed directly at the piano keys, a design choice made with dyslexic learners in mind so no sheet music required.
  * [FoldedNotes](https://github.com/himanijha/FoldedNotes/tree/main) is an anonymous peer support hotline that creates an emotional support community and uses haptic hardware that plays different vibration patterns for different emotional states.
* **Giving back through tech for nonprofits.**
  * I led development on the [Save Cantonese Volunteer Roster](https://github.com/himanijha/VolunteerRoster), which flags inactive volunteers from Zoom and Slack logs, cutting down the manual tracking work organizers used to do by hand.
  * I helped build a volunteer management system for Alzheimer's San Diego, tracking and analyzing data for 100,000+ volunteers.
  * I served as dev lead on [Mending Kids](https://github.com/novaforgood/mending-kids), building an inventory system that helps keep their surgical supplies organized for 6,000+ child surgeries worldwide.
